    Objetivo

    el objetivo de crear mi archivo estilos.css es para separar al estructura del diseño.

    • asi tendremos mi archivo .htmlsolo para estructura elemntos etiquetas y contenido.
    • y mi archivo .csssolo para los selectores y propiedades de diseño que le asignare a cada etiqueta.

    Modo de Uso

    para relacionar un archivo html con su archivo css (hojas de estilo). debemos hacer uso de nuestra etiqueta link en la cabezera del documento. y en su atributo href le indicamos la ruta donde se encuentra el archivo
